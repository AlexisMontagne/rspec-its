# Rspec::Its

RSpec::Its provides the `its` method as a short-hand to specify the expected value of an attribute.

## Installation

Add this line to your application's Gemfile:

    gem 'rspec-its'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rspec-its
